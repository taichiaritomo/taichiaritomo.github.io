Demo!
In a research seminar at Rutgers, I built a simple tool that models visual interest in textual information. It continuously processes text (left) as an image and generates its saliency map (right). The intensity of white corresponds to the likelihood that a human viewer will gaze at that point first, according to a computer vision algorithm that models visual attention.

I made this tool with the hypothesis that it could help us better understand the audience's experience of seeing the information that we design.

Built with Javascript and QuillJS, a text-editor API
