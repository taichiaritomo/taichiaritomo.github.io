On the Artists Space website, there is a selection input that lists every month since its opening in 1972. The trailing edge of those month-words oscillate with every year. The wavy sun was an accidental glitch, but it turned out to be better left unfixed...

Every few weeks, the Openstreetmaps community packages all of the OSM world data into a "planet mirror" file. This data includes 81,000 picnic tables. I've seen picnic tables on trail maps, but not on city maps. This website places a marker on a google map for each picnic table. The canonical marker symbol looks like a flower petal, so clusters of markers are flowers.
